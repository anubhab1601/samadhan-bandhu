- Uploads geo-tagged project site photos
- Uploads supporting documents
- Submits application
- Views application details
- Tracks application progress

### 3. **State Review Workflow** ✅
- State receives applications
- Reviews application details
- Forwards to IVA for verification
- OR Rejects with remarks

### 4. **IVA Verification Workflow** ✅
- IVA receives applications
- Conducts village visit
- Verifies population & eligibility
- Assesses infrastructure
- Checks community participation
- Verifies financial details
- Submits verification report
- Recommends Approve/Reject

### 5. **Center Consent & Fund Release Workflow** ✅
- Center receives IVA-verified applications
- Reviews complete application
- Views IVA verification summary
- Approves/Rejects consent
- Releases funds in installments
- Tracks fund utilization

### 6. **Field Officer Inspection Workflow** ✅
- Field Officer assigned to project
- Conducts on-site inspection
- Assesses work progress
- Checks quality & safety
- Verifies financial expenditure
- Uploads geo-tagged inspection photos
- Submits inspection report

### 7. **Agency Tender & Execution Workflow** ✅
- Agency browses available tenders
- Views tender details
- Applies for tenders
- Gets selected
- Manages project execution
- Tracks milestones & progress
- Submits monthly progress reports
- Uploads progress photos & bills
